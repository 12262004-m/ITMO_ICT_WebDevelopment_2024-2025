Реализация клиентской части приложения средствами vue.js.

Описание проекта: система предназначена для администрирования спортивного комплекса (под спортивным комплексом подразумевается, например, спортивная школа). Она обеспечивает учет данных о секциях, тренерах, учениках и расписании тренировок. Спортивный комплекс включает в себя залы для разных видов спорта.

В ней представлены следующие интерфейсы 1. Регистрация и авторизация для администраторов - Форма для вода данных

  1. Главная страница.
  2. Расписание тренировок на сегодняшний день во всех залах

  3. Новостная лента

  4. Навигационная панель

  5. Вкладки: расписание, ученики, тренеры, отчет, секции

  6. Ученики.

  7. Список все учеников (ФИО, пол, дата рождения, контактные данные, дата регистрации)
  8. Добавление учеников

  9. Редактирование учеников

  10. Тренеры.

  11. Список всех тренеров (персональная информация)

  12. Возможность добавлять новых тренеров в базу.

  13. Расписание.

  14. Актуальная информация о занятости спортзалов по дням недели

  15. Фильтрация по тренеру

  16. Секции.

  17. Список всех секций

  18. Добавление ученика в конкретную секцию

  19. Отчет по тренерам

  20. Формирование отчета про конкретного тренера
